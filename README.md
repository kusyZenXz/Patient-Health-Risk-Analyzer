## 📌 Overview

This project is a simple healthcare analytics model built using Python.
It collects patient health data, calculates BMI, classifies risk level, and stores the data for further analysis.

The goal of this project is to move beyond basic coding and start thinking in terms of **real-world data problems**.

---

## 🎯 Problem Statement

In clinical settings, identifying high-risk patients early is important.
Basic indicators like BMI and blood sugar can provide quick insights.

This project simulates a basic system that:

* Evaluates patient health metrics
* Categorizes risk levels
* Stores structured data for analysis

---

## ⚙️ Features

* 📥 User input for multiple patients
* 🧮 BMI calculation
* ⚠️ Risk classification (Low / Medium / High)
* 📁 Data stored in CSV file
* 📊 Basic summary analysis

---

## 🛠️ Tech Stack

* Python
* CSV handling
* Conditional logic
